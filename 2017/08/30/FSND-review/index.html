Udacity FSND课程回顾

Intro

FSND全称Full Stack Web Developer Nano Degree，这里的全栈指的是前端的网页开发和服务器后台开发。

总体框架

这门课因为内容比较多，容易让人感觉很凌乱，东一块西一块，不好把握。特别是网站改版之前，很难把握其脉络。

网站改版之后，将所有的学习内容分成了五个部分：

  1. Programming Fundamentals and the Web
  2. Web Applications and Development
  3. The Backend: Database & Applications
  4. The Frontend: JS & AJAX
  5. Deploying to Linux Servers

结构非常清楚了。其中1和4可以算作是前端的内容，2和3可以算作后端内容，1和2是基础，3和4属于相对高级的内容。最后一部分讲在服务器上的部署。

而且，这门课程因为是google赞助的，所以涉及到google产品的内容比较多。比如，第2部分中，学习了Google Cloud Platform平台的使用和开发，其他还有Google Maps API，Google Oauth等等。
