使用Jupyter Notebook作为linux下的markdown编辑器

2017-11-08 • 分类: Tool • 标签: Markdown, jupyter notebook

Intro

之前在mac系统下用codeRunner来写markdown，结果差强人意。Linux平台下一直没有找到很满意的markdown编辑器，typora还没有linux版本，remarkble和retext有各种小问题，sublime text舍不得买license，虽然可以一直用，但完美主义又出来作祟。

今天灵机一动，用来写代码的jupyter notebook也可以写markdown啊！而且可以少安装一个软件。
