Intro

人工智能和机器学习在最近几年出现井喷式发展，私以为机器学习非常有可能成为将来程序员的必备技能，这是时代对我们提出的新要求。而吴恩达教授在coursera上教授的Machine Learning课程，可以说是入门机器学习的最佳选择。我在2016年完成了此门课程的学习，但在后续课程的学习过程中，发现自己对这门课程的掌握并不牢固。本文将对自己的学习过程和所得进行简单的总结，希望借此可以对课程有一个更好的把握。

数据来源

images

下载ILSVRC2012的的数据集 中任务1&2的训练图片（138GB)，将其中与食品有关的压缩包提取出来。

将提取出来的压缩包中包含的图片解压到一个统一的目录下。

bounding boxes

下载任务1&2的bbox数据，同样将其解压到同一个目录下。

由于并不是所有的图片都有相对应的bbox数据，写了一个shell脚本将图片分成了with_bbox和without_bbox两部分，后续只使用包含了bbox的图片。

dat file && list file

下载任务1&2的Development kit，将其中的mat文件复制到到我们的data目录下，改名为meta_det.mat。

接下来我们需要所有训练数据的一个列表文件，即/DATA_DIR/data/det_lists/train.txt。写一个shell脚本即可实现，注意列表不包含后缀名。
