学习经验和总结-机器学习

2017-07-27 • 分类: ML • 标签: Deep Learning, Machine Learning

人工智能和机器学习最近几年出现井喷式发展，近来更是成为全社会关注的焦点，我也有幸成为了学习大军中的一员。

机器学习的局限性

随着学习的不断深入，逐渐意识到现在机器学习的局限性。现在的机器学习距离真正的人工智能还差得很远。

当下机器学习特别是监督学习的本质，是建立一个模型，并有效地学习了大量数据来对模型进行了调整和拟合。这样当遇到新的此类问题的时候，就可以根据模型来进行预测。

如果接下来几年时间不能继续取得突破性进展，人工智能可能会成为下一轮的泡沫。

Update2018-02-07:

之前对机器学习的认识有些偏颇，机器学习不只是发展强人工智能这一个目标。对于不能靠hard code解决或解决不好的一类问题，机器学习都提供了很好的解决方案，比如计算机芯片中的分支预测功能（比如Ryzen），还有去年Google提出的用机器学习索引来代替B-Tree（论文）等等。机器学习甚至可以提高到“第二类”编程方法的高度。

我学习的主要课程

  • 机器学习， Coursera

    吴恩达教授的这门课程，是入门的不二之选。课程并没有深入讲解热门的深度学习，而是专注于机器学习的基础理论和分析。通过学习这门课程，可以获得机器学习的方法论知识，从大局上来更好地把握。

    课程采用Octave语言，作业很经典，一定要完成。

  • CS231n，斯坦福大学

    计算机视觉课程，也对RNN进行了讲解。

    课程采用python语言，包含三个作业，基本将机器学习的各个组件都实现了一遍。如果掌握充分，甚至可以在此基础上实现一个简单的机器学习框架了。最后一个作业包含了GAN的内容。

  • Deep Learning，Udacity

    这门课程是Google制作的，特点是短小精悍。视频时间很短，丝毫不浪费笔墨。

    课程也包含了四个作业，因为是Google制作的课程，肯定是使用Tensorflow框架，刚好与CS231n课程作业形成互补，可以专注于问题的解决而非框架的实现。

  • CS224n， 斯坦福大学

    自然语言处理课程。

    作业内容和难度相对课程内容要简单很多。但最后的Project是做阅读理解（阿里NLP团队最近得到了超过人类的成绩），难度应该不小，我还没有做。

  • CS294，加州大学伯克利分校

    强化学习课程，从2017年秋季开始，课程进行了比较大的改动，讲解更加详细，内容更丰富。

    目前（2018年2月初）刚开始学习。此前只有一点DQN方面的知识，所以这门课有一定难度。

    最近Alpha Zero可以说是非常惊艳，但DeepMind在星际争霸2项目上遭遇到了瓶颈，也体现出了当前由弱人工智能向强人工智能发展的无力。
